# Vocabee — продуктова документація (v2)

Жива документація застосунку Vocabee: поточна поведінка коду ([ЗАРАЗ]) і затверджені зміни ([НОВЕ]/[МАЙБУТНЄ]).
На основі цих документів промальовується дизайн нових флоу та UI-елементів.

> **Почни звідси:** [FUNCTIONAL-OVERVIEW.md](FUNCTIONAL-OVERVIEW.md) — майстер-інвентар усього функціоналу за доменами з посиланнями на профільні доки.
> Джерело істини для нової поведінки — **журнал рішень D1–D10** у [00-overview-and-decisions.md](00-overview-and-decisions.md).
> Кожен пункт документації явно позначений: **[ЗАРАЗ]** — як працює зараз у коді, **[НОВЕ]** — затверджена зміна, **[МАЙБУТНЄ]** — заплановано пізніше.

## Ключові терміни (коротко)

- **Монетка (beecoin)** — внутрішня валюта. Старт 50; +10 за рекламу; промо-бонуси зверху. Витрати: словник понад 2 безкоштовні = −10, пошук перекладу = −1.
- **Анонім** — без акаунта: безкоштовний пробний режим (2 словники / 50 слів), **без монеток** (D2).
- **Авторизований** — вхід через Google: монетки, реклама, промо, синхронізація, бекап.
