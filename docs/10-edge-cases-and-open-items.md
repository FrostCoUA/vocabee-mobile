# 10. Крайові випадки і відкриті питання

> Реєстр error-станів, граничних умов, відкритих припущень і backlog для дизайну.
> Кожна поведінка позначена: **[ЗАРАЗ]** — як працює в поточному коді, **[НОВЕ]** — затверджена зміна (журнал рішень D1–D9 у [00-overview-and-decisions.md](00-overview-and-decisions.md)).
> Де є посилання на код — формат `file:рядок`.

## 2. Відкриті припущення (потребують уточнення)

Кожен пункт — **відкрите рішення**; познач власника продукту й дедлайн при груммінгу.

| # | Питання | Поточний стан коду | Варіанти / що уточнити |
|---|---------|--------------------|------------------------|
| O1 | **premium-tier: лишити як майбутнє чи прибрати?** | `UserTier = 'anonymous' \| 'registered' \| 'premium'` (`user-tier.ts`). `TIER_MAX_RESULTS` = **50/50/50** — premium **ідентичний** registered. Колонка `users.isPremium` існує (`users.ts:25`), але **ніде не виставляється в true**. Коментар у коді: «Premium stays declared so the slot is wired in». | (a) Лишити заглушку під майбутню монетизацію (рекомендовано — слот уже всюди прокинуто); (b) прибрати tier і колонку, щоб не вводити в оману. Рішення власника продукту. |
| O2 | **isAnonymous-прапорець / RegisteredUserGuard — мертві чи зарезервовані?** | `RegisteredUserGuard` (`registered-user.guard.ts`) перевіряє `user.isAnonymous`, але **не застосований на жодному роуті** (grep по `@UseGuards` його не використовує). `users.isAnonymous` defaults `false`, у `loginWithGoogle`/`register` завжди `false` (`auth.service.ts:63,114`). Анонім у Vocabee = **відсутність рядка в БД** (`tierFromUserRow(null)`), а не `isAnonymous=true`. | Уточнити: чи планувались серверні анонімні сесії (guest-токени)? Якщо ні — `RegisteredUserGuard` і `isAnonymous` **мертвий код**, прибрати або задокументувати як зарезервоване. Розузгодження з моделлю D2 (анонім = без акаунта на клієнті). |
| O3 | **Анонімний кеп результатів пошуку: 50 для всіх vs «3 для аноніма»** | **Конфлікт:** код `TIER_MAX_RESULTS` = 50/50/50 (`user-tier.ts`); але `ApiOperation` опис того ж ендпоінта каже **«anonymous max 3, registered 5, premium 10»** (`lexicon.controller.ts:33-35`).
