# Vocabee — Design Brief (самодостатній, для Claude Design)

> Цей файл містить **усе потрібне**, щоб спроєктувати / апгрейднути дизайн застосунку Vocabee. Читати репозиторій не треба — весь контекст тут. Мова застосунку — **українська**. Стиль написання тексту в UI — **речення з малої літери** (sentence case), без ALL CAPS і Title Case.

---

## 0. Що проєктуємо (мета)
**Vocabee** — мобільний застосунок для вивчення іноземних слів. Сигнатурний цикл: **читаю → зберігаю слово (текст/голос + AI-переклад) → організовую в тематичні словники → тренуюсь**. Бджолина тематика; внутрішня валюта — **монетки (beecoins)**.

Уже є робочий прототип (Android, Compose). Завдання дизайну — **підняти прототип до цілісного, відполірованого продукту** і домалювати **нові флоу й UI-елементи** (промо-система, мерж при вході, розумне тренування, інлайн-довідка, теми-іконки, анімації видалення тощо). Нижче — повний перелік того, що треба намалювати, зі станами, копірайтом і специфікаціями руху.

> Стартова візуальна референс-точка вже намальована в чаті: банер-стани, промо-ботомшит, мерж-шторка, дерево онбордингу, шторка створення словника, тултіп, лідерборд, конвеєр відбору тренування, картка+підсумок сесії. Тримай їхній стиль як основу й апгрейдь.

### Типографіка
- Шрифт **Manrope** (геометричний, дружній, повна кирилиця). Два-три ваги: Regular, SemiBold/Bold, ExtraBold для заголовків/чисел.
- Масштаб (орієнтир): заголовок екрана 28–34sp, заголовок шторки 20sp, заголовок картки 16sp, тіло 14–16sp, дрібний/підпис 12–13sp. Числа балансу/прогресу — ExtraBold.

### Форма й поверхні
- **Плоско**, без градієнтів, шуму, важких тіней. Тіні лише функціональні (легке підняття активного елемента).
- Радіуси: картки `16–22`, банери `18`, bottom sheets `28` (верхні кути), кнопки-пігулки `16`, чипи `11–14`, бейджі `13–15`.
- Іконки — **лінійні** (outline), 2.1–2.4 товщина штриха, 18–24px.
- Багато повітря, один чіткий головний акцент на екрані.

### Тон
Дружній, теплий, бджолиний; короткі підбадьорливі формулювання. Без канцеляриту. Емодзі — мінімально (можна зрідка як акцент: 🎉 у підсумку).

## 2. Архітектура й навігація
- Single-screen застосунок із **нижнім таб-баром (3 таби):** **Словники** (стартовий) · **Тренування** · **Профіль**.
- Повноекранні флоу (онбординг, авторизація, додавання слова) — **над** таб-баром, ховають його.
- Bottom sheets — для дій без переходу (створення словника, видалення, вибір мови, промо, мерж, довідка «?»).

### Два стани користувача (впливає на UI)
| | **Анонім** (без акаунта) | **Авторизований** (Google) |
|---|---|---|
| Ліміти | 2 словники, 50 слів | 2 безкоштовні словники + платні |
| **Монетки** | **немає** | є (старт 50) |
| Реклама/промо | **немає** | є |
| Банер на головній | «гостьовий режим» | гаманець / промо / критичний |
| Синхронізація | лише локально | повна |

> Ключове: **анонім не бачить монеток, реклами й промо** — у нього банер гостьового режиму й заклик зареєструватися. Реєстрація — момент апгрейду (вмикає всю економіку).

## 3. Економіка монеток (що показуємо в UI)
- Старт **50** монеток (після входу). Реклама **+10**. Промо — бонуси **зверху**.
- Витрати: **−10** за словник понад перші 2 безкоштовні; **−1** за пошук перекладу.
- Критичний баланс **≤3** → попереджувальний банер/стан.
- Іконка монетки — стільник/Sparkle/`coins`. Бейдж балансу: іконка + число.

## 4. Екрани та їхні стани (що намалювати)

> Для кожного екрана намалюй усі перелічені **стани**.

### 4.1 Splash
Лого voca**bee** (жовтий акцент «bee») на фіолетовому радіальному фоні, підпис «Збирай слова. Будуй словники.», знизу «MADE WITH CARE». Авто-перехід ~1.9с.

### 4.2 Онбординг (3 слайди)
Фіолетовий фон, ілюстрація + заголовок + підзаголовок, індикатор-крапки, «Пропустити» / «Далі»→«Почати». Теми слайдів: «Зберігай слова під час читання», «Створюй тематичні словники», «Повторюй, коли зручно».

### 4.3 Авторизація (реальний Google-вхід — НОВЕ)
Email+пароль, «Продовжити з Google», «Продовжити з Facebook», тогл реєстрація↔вхід, «Пропустити». **Стани:** звичайний, лоадинг входу, помилка.

### 4.4 Вибір мов
Дві секції «Я розмовляю» / «Я вивчаю», сітка карток мов (прапор+назва), вибрана з галочкою, угорі пігулки пари з стрілкою. Кнопка «Готово». Інваріант: дві мови різні.

### 4.5 Словники (Home)
- **Порожній стан:** ілюстрація + «Створи свій перший тематичний словник» + кнопка.
- **Зі словниками:** сітка 2 колонки карток (іконка теми, назва, лічильник слів, колір), FAB «+», свайп-видалення картки.
- **Банер-слот** угорі (один) — стан залежить від користувача (див. §5).
- Хедер: «Словники» + метрики (N словників · M слів зібрано).

### 4.6 Деталь словника
- Хедер: назва + лічильник, **перемикач напрямку перекладу** (впливає на STT — поряд «?»).
- Список слів, **згрупованих за вихідним словом**; кожен рядок: оригінал, IPA, переклад, кнопка озвучення (TTS), розгортувані деталі (значення/синоніми/антоніми/форми/приклади з AI-тегом), кільце знання %.
- Свайп-видалення рядка (+ Undo). Закріплена кнопка «+ Додати слово».
- **Стани:** порожній (підказка до «+»), зі словами.

### 4.7 Додавання слова (повноекранний оверлей — сигнатурна взаємодія)
Морф із пігулки «+ Додати слово» на весь екран. **Стани:**
- **Idle:** поле вводу зверху + велика кнопка мікрофона по центру.
- **Recording:** анімована хвиля (orange), «Слухаю…», «торкнись, щоб зупинити».
- **Loading:** спінер «Шукаю…».
- **Error:** «не вдалося розпізнати / немає перекладу» + повтор.
- **Результати:** список рядків (вихідне слово, IPA, переклад, кнопка `+`/`✓` якщо вже додано, AI-атрибуція). Дебаунс пошуку ~1с.

### 4.8 Тренування (Practice)
- **Setup:** мультивибір словників (горизонтальні чипи), «Почати». Стан «нема слів».
- **Картка (flip):** передня грань — вихідне слово + чип теми + IPA + кільце знання; задня — переклад на кольоровому тлі. Тап = переворот.
- **Кнопки:** «Не знаю» / «Знаю»; після «Не знаю» — авто-показ відповіді + одна кнопка «Далі».
- **Прогрес:** лічильник «4 / 10 · правильно N» + сегментований бар.
- **Done:** «8 з 10 правильно», статистика (+знань %, стрік, нові), «Ще раз» / «До словників».

### 4.9 Профіль
Картка акаунта (аватар/ім'я/email) або картка входу для аноніма; «Я розмовляю» / «Я вивчаю»; тогл темної теми; тогл сповіщень; рядок статистики (стрік/слова/тести); «Допомога», «Запросити друзів»; «Вийти»; версія.

### 4.10 Лідерборд (НОВЕ)
Топ-10 тижня за переглядами реклами: рядки (місце, аватар, ім'я, к-сть переглядів), перші 3 з медалями, закріплений рядок «Ви: N місце», бейдж «+50 для топ-10», зворотний відлік до скидання (щопонеділка), кнопка «Дивитись рекламу — піднятись вище».

## 5. Банер монеток — усі стани (НОВЕ, центральний елемент)

Один банерний слот на головній. Намалюй усі стани:

| Стан | Кому | Вигляд / копірайт |
|---|---|---|
| **Гостьовий режим** | анонім | purple/tint, іконка user: «Гостьовий режим» · «Словники 1/2 · слова 12/50. Далі потрібен Google акаунт.» |
| **Ліміт гостя вичерпано** | анонім | orange/peach: «Ліміт гостя вичерпано» · «Увійди, щоб продовжити.» |
| **Гаманець (фолбек)** | авторизований | бейдж балансу + «Переклади за монетки» · «1 пошук коштує 1 монетку. Подивись відео і отримай +10.» |
| **Критичний (≤3)** | авторизований | orange: «Монетки майже закінчились» · «Лишилось 3 монетки. Відео дасть +10.» |
| **Промо · milestone (active)** | авторизований | gift, прогрес-бар: «Бонус за перегляди» · «Сьогодні 4 з 10 реклам — далі +20 монеток.» |
| **Промо · claimable** | авторизований | green: «Готово! Забери +20» · «10 з 10 реклам — натисни, щоб отримати приз.» |
| **Промо · daily_streak** | авторизований | flame + точки днів: «Стрік 3 з 5 днів» · «Сьогодні 4 з 3 — день зараховано.» |
| **Промо · leaderboard** | авторизований | trophy: «Тижневий топ-10» · «Ви 14-й. Топ-10 за переглядами отримають +50.» |

**Пріоритет показу:** анонім→гостьовий; інакше критичний(≤3)→промо(найвищий priority)→гаманець. Клік по банеру → відповідний bottom sheet.

## 6. Promo-система (config-driven) — bottom sheet (НОВЕ)
Банер → ботомшит із поясненням і прогресом. Намалюй патерн + 4 типи:
- **Структура:** бейдж винагороди (+N) → заголовок → пояснення (1–3 речення) → **візуалізація прогресу** → кнопки.
- **Візуалізації прогресу за типом:** `milestone` — прогрес-бар «4 з 10»; `daily_streak` — **точки днів (5)** заповнені/порожні + міні-бар «сьогодні 4 з 3»; `one_time` — стан «готово»; `leaderboard` — місце + відлік.
- **Кнопки (з конфіга):** `Дивитись рекламу` (primary, запускає rewarded ad), `Забрати +N` (claimable), `Дивитись рейтинг` (deeplink), `Увійти` (locked-для аноніма), `Пізніше`.
- **Стани шита:** active (прогрес + дивитись рекламу) · claimable (підсвітка green, забрати приз) · completed (виконано, відлік до скидання) · locked (для аноніма — увійти).
- **4 кампанії:** 10 реклам→+20 (повторювана); 3 реклами/день × 5 днів→+50; реєстрація→+50 (разово); тижневий топ-10→+50.

## 7. Нові шторки/елементи (НОВЕ) — намалювати

### 7.1 Створення словника
Поле назви → **пікер іконок теми** (сітка 5×3, 14 іконок: серіал/фільм, книга, подорожі/літачок, їжа, робота, школа, спорт, музика, природа, техніка, шопінг, діти, здоров'я, загальна) → ряд **кольорів** (8) → **мова словника** (дефолт із профілю + «Змінити» = оверрайд пари) → **цінник** («3-й словник · спишеться 10 монеток», якщо платний) → «Створити словник».

### 7.2 Видалення словника (підтвердження)
«Видалити словник? · І N слів буде видалено · Цю дію не можна скасувати», «Монетки за створення не повертаються». Кнопки «Скасувати» / «Видалити» (red). Після — **Undo-снекбар 6с**.

### 7.3 Видалення слова
Без модалки — свайп → видалення + **Undo-снекбар 6с** «Слово видалено · Скасувати».

### 7.4 Sync-мерж при вході (НОВЕ, важливо)
«Є два стани акаунта» → дві картки (на телефоні N словників/слів · на бекенді M) → плашка вартості «Мерж спише 10 монеток · 1 зайвий платний словник · баланс 48 → 38» → варіанти: **Злити все разом (−10)** (primary), «Затерти серверні дані локальними», «Лишити серверне, відкинути локальне», «Скасувати вхід», лінк «Увійти іншим email». Якщо монеток бракує — лишаються лише не-мерж-варіанти.

### 7.5 Інлайн-довідка «?»
Маленька кругла іконка `?` біля неочевидних місць (монетки, ліміти, напрямок перекладу, мова словника, синхронізація, промо, видалення). Клік → bottom sheet: іконка + заголовок + пояснення (2–4 речення) + опційна кнопка дії («Дивитись рекламу» / «Увійти»). Приклад «Що таке монетки?» з блоками «заробляєш / витрачаєш».

### 7.6 Потрібні монетки / Потрібен акаунт
Шторки-гейти: «Потрібні монетки» (перші 2 словники безкоштовні, далі 10; кнопка «Відео за +10») і «Потрібен акаунт» (для аноніма за лімітом; «Увійти Google»).

### 7.7 Undo-снекбар, стан «0 монеток»
Снекбар із дією «Скасувати» (для слів і словників). Окремий порожній стан «монеток немає» з проактивним «заробити».

## 8. Ключові флоу (намалювати як послідовності)
1. **Онбординг-розвилка:** Splash → (онбординг пройдено? ні→слайди→**реальний Google-вхід**) → якщо увійшов і на сервері є дані → **пропуск вибору мов** (мови/баланс із сервера); якщо новий акаунт → вибір мов→push; якщо скіп → вибір мов локально → Main.
2. **Заробити→витратити монетки:** банер/гейт → дивитись рекламу (+10) → баланс росте → створення словника/пошук перекладу списує.
3. **Промо-клейм:** банер → ботомшит (прогрес) → дивитись рекламу → перехід active→claimable → забрати приз.
4. **Вхід із мержем:** Google-вхід → є локальні+серверні дані → мерж-шторка з вартістю → вибір.
5. **Видалення + Undo:** свайп слова/словника → (для словника — підтвердження) → елемент зникає з анімацією → Undo-снекбар → відновлення або фіналізація.
6. **Сесія тренування:** вибір словників → 10 карток (переворот, Знаю/Не знаю) → прогрес → підсумок → «Ще раз».

## 9. Motion & взаємодії (специфікації)

### Токени
- Easing standard `cubic-bezier(0.2,0,0,1)`; emphasized — з легким overshoot/spring.
- Тривалості: S 120–180мс (мікро), M 260–320мс (поява/reflow/прогрес), L 420–460мс (переворот, морф).
- Haptic: light (натиск, поріг свайпу, «Знаю»), medium («Не знаю», коміт видалення).
- Кожна руйнівна дія має **зворотний хід** (undo / контр-анімація).

### Переворот картки тренування
`rotateY 0→180`, L, **spring** із мікро-overshoot; підйом картки `scale 1→1.04→1` + тінь; кросфейд контенту у вікні 80–100°; зміст задньої грані контр-обертається (не дзеркальний); idle-погойдування ±4° (натяк «торкнись»); haptic на перетині 90°.

### Флов «Не знаю»
Натиск → червоний пульс по картці + haptic medium → **авто-переворот** на відповідь → кільце знання вниз −20% (M, count-down) → **дві кнопки морфляться в одну «Далі»** (S) → слово **перечерговується** в кінець колоди → тап «Далі» → поточна картка йде вліво (fade), наступна входить справа. Контраст: **«Знаю»** — зелена галочка-спалах, без показу відповіді, одразу далі, кільце вгору.

### Прогрес
Сегментований бар (10 поділок), анімоване заповнення M, count-up «правильно», колірна мапа (зелено/помаранч), наприкінці → spring-pop у Done.

### Свайп-видалення (слово)
Прогресивний реквіл (іконка масштабується), поріг ~50% (+haptic), **full-swipe-to-delete** при далекому свайпі, **колапс рядка по висоті** + reflow списку (M), **Undo-снекбар 6с**, rubber-band за межею.

### Свайп-видалення (словник)
Реквіл → **шторка-підтвердження** (бо каскад) → коміт → картка згортається (scale+fade) + reflow сітки → **Undo-снекбар** + примітка про монетки. Альтернатива — довге утримання → контекст-меню.

### Морф «Додати слово»
Пігулка морфиться на весь екран (~420мс); поле з'їжджає згори, мікрофон з'являється по центру (стагер 50–80мс); `+` на рядку → пульс + галочка + слово «падає» в словник.

## 10. Копірайт для нових поверхонь (готові рядки, українською)
- Гостьовий банер: «Гостьовий режим» / «Словники {n}/2 · слова {m}/50. Далі потрібен Google акаунт.»
- Гаманець: «Переклади за монетки» / «1 пошук коштує 1 монетку. Подивись відео і отримай +10.»
- Критичний: «Монетки майже закінчились» / «Лишилось {n} монеток. Відео дасть +10.»
- Промо milestone: «Бонус за перегляди» / «Сьогодні {x} з 10 реклам — далі +20 монеток.»
- Промо streak: «Стрік {d} з 5 днів» / «Сьогодні {x} з 3 — день зараховано. Тримай ритм!»
- Реєстрація-бонус: «Подарунок за реєстрацію» / «Заберіть +50 монеток за вхід через Google.»
- Лідерборд: «Тижневий топ-10» / «Ви {r}-й. Топ-10 за переглядами отримають +50.»
- Мерж: «Є два стани акаунта» / «Мерж спише {c} монеток · {k} зайвий платний словник.»
- Undo: «Слово видалено · Скасувати» / «Словник видалено · Скасувати».
- Тултіп монеток: «Монетки (beecoins) — внутрішня валюта. На старті 50. Заробляєш +10 за рекламу, витрачаєш −10 за словник (понад 2) і −1 за пошук перекладу.»
- Тренування done: «{c} з 10 правильно» / «Гарна робота! Слабкі слова повернуться згодом.»

## 11. Перелік дизайн-деліверіблів (чекліст)
**Апгрейд наявних:** Splash · онбординг · авторизація (зробити «справжньою») · вибір мов · Словники (порожній/зі словниками) · Деталь словника · Додавання слова (4 стани) · Тренування (картка/прогрес/done) · Профіль · Створення словника · Підтвердження видалення.

**Намалювати нове:** банер монеток (8 станів) · промо-ботомшит (4 типи × стани) · лідерборд · мерж-шторка з вартістю · пікер іконок тем + оверрайд мов · інлайн-«?»-ботомшит (патерн + кілька прикладів) · Undo-снекбар · стан «0 монеток» · гейти «Потрібні монетки» / «Потрібен акаунт» · перемикач напрямку перекладу · анімації (переворот, «Не знаю», свайп-видалення, морф додавання).

**Принципи скрізь:** плоский бренд Vocabee (§1), sentence case, один головний акцент, два стани користувача (§2), кожна руйнівна дія з undo, рух підказує наслідок.
